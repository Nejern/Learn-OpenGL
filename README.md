# Learn OpenGL

## Control

- `W/A/S/D` - movement
- `Mouse` - camera rotate
- `Scroll` - zoom
- `SPACE` - stop/resume time
- `C` - open menu
- `L` - polygon mode
- `F11` - fullscreen mode

## Compile

1. Install libs

- glfw
- assimp
- glm
- stb

2. Make build dir

```
mkdir build
cd build
```

3. Setup CMake and compile

```
cmake ..
make
```
